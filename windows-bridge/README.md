# Мост для Kaspi Донатов (Windows + Связь с телефоном)

Этот скрипт позволяет вашему ПК читать уведомления от приложения "Связь с телефоном" (Phone Link) и автоматически отправлять донаты из Kaspi.kz прямо в ваши виджеты OBS.

## Как это работает?
1. Зритель отправляет вам перевод в Kaspi.kz.
2. На ваш телефон приходит пуш-уведомление.
3. Приложение "Связь с телефоном" дублирует это уведомление на ваш ПК (Windows 10/11).
4. Этот скрипт мгновенно читает уведомление, достает сумму, имя и сообщение, и отправляет их в ваш виджет.

## Инструкция по установке

### Шаг 1: Установите Python
1. Скачайте Python с официального сайта: [python.org/downloads](https://www.python.org/downloads/)
2. При установке **ОБЯЗАТЕЛЬНО** поставьте галочку **"Add Python to PATH"** (Добавить Python в PATH).

### Шаг 2: Установите нужные библиотеки
Откройте командную строку (нажмите `Win + R`, введите `cmd` и нажмите Enter) и выполните команду:
```cmd
pip install winsdk requests
```

### Шаг 3: Настройте "Связь с телефоном"
1. Убедитесь, что ваш телефон подключен к ПК через стандартное приложение Windows "Связь с телефоном".
2. Убедитесь, что уведомления от Kaspi.kz приходят на ПК (всплывают в правом нижнем углу).

### Шаг 4: Запуск
1. Скачайте файл `kaspi_listener.py` на свой компьютер (например, на Рабочий стол).
2. Откройте командную строку (`cmd`) в папке с файлом.
3. Запустите скрипт командой:
```cmd
python kaspi_listener.py
```

### Шаг 5: Разрешение доступа
При первом запуске скрипт может попросить доступ к уведомлениям. 
Перейдите в настройки Windows: **Параметры -> Конфиденциальность и защита -> Уведомления** и разрешите доступ для Python.

## Всё готово!
Теперь просто держите это черное окошко (консоль) открытым во время стрима. Как только придет донат, скрипт напишет об этом и отправит алерт в OBS!
